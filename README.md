# Survey Insights Dashboard – Power BI

An interactive Power BI dashboard designed and developed for visualizing and analyzing survey responses across multiple categories with sentiment breakdown and actionable insights.

## 📊 Project Overview

This project showcases a comprehensive data analytics solution that transforms raw survey data into meaningful visual insights. The dashboard enables stakeholders to explore survey feedback across services, staff, and facilities with dynamic filtering and interactive visualizations.

## 🚀 Getting Started

### Prerequisites
- Microsoft Power BI Desktop (latest version)
- Access to Power BI service (optional, for sharing and collaboration)

### Usage

1. **Open the Dashboard**
   ```
   Open SurveyDashboard.pbix in Power BI Desktop
   ```

2. **Explore the Data**
   - Use slicers to filter by categories and date ranges
   - Hover over visualizations for detailed tooltips
   - Click on visual elements for cross-filtering

3. **Customize for Your Data**
   - Replace sample data with your own survey data
   - Update data connections in Power Query
   - Refresh the dashboard to reflect new insights

### Connecting Your Data

1. Open Power BI Desktop
2. Go to "Get Data" → "CSV" or your data source
3. Load your survey data
4. Map fields to existing Power Query transformations
5. Publish to Power BI Service (optional)
